SceneLoaderクラス

概要

シーン遷移のクラス。UnityEngine.SceneManagement.SceneManager.LoadScene()の
代わりに使うことでシーン遷移の際フェードイン/フェードアウトを行います
※UnityEngine.SceneManagement.SceneManager.LoadScene()と併用しないでください。


変数

変数名 変数型 説明 　　　　　　　　　　　
IsFade bool フェード中か否か? 　　　　　
Fadeings float フェードどのくらいか?
(フェードアウト開始時 = -1)
(フェードイン終了時 = 1)
(シーン遷移時 = 0)
FadeColor Color フェードする色
FadeInTime float フェードインの時間(/sec)　
FadeOutTimefloat フェードアウトの時間(/sec)

関数

引数はint型,string型。string型はシーン名、int型はシーンの番号

関数名 戻り値 引数 説明
LoadN void -- フェードイン/フェードアウトなしの遷移
Load void -- 単色のフェードイン/フェードアウト
Load_DXvoid -- ちょいおしゃれな遷移(未実装)
